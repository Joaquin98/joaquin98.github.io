How to install

Chrome Browser

1. Extention Tampermonkey 2. Grepobot userscript

1. In order to use Grepobot you need to install an extention named Tampermonkey. Tampermonkey is an userscript manager for Google Chrome.

2. Once this extention is installed you have to download and install our userscript..

Morzilla Firefox

1. Extention Greasemonkey 2. Grepobot userscript

1. In order to use Grepobot you need to install an extention named Greasemonkey. Greasemonkey is an userscript manager for Morzilla Firefox.

2. Once this extention is installed you have to download and install our userscript.

1. Standalone Extention Grepobot

1. You can also install our standalone extention. When you have downloaded the install.xpi file you have to locate the file and drag&drop the file into your Firefox Browser.

When its installed you have to restart the browser in order to initialize Grepobot.
